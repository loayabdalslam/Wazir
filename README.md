# Wazir 🌍🧠

**Wazir** is an advanced Multi-Agent National Simulation Platform (v3.3.0). Created by **Loaii abdalslam**, it leverages a fully autonomous **Swarm Intelligence** cabinet of 10 specialized AI ministers—including the President, Economy, Defense, Health, and Intelligence—to simulate dynamic national governance.

By grounding its analysis in real-time World Bank statistics and live web data, the swarm synthesizes hyper-realistic scenarios, statistical projections, and unpredictable wildcards tailored to any country and geopolitical goal.

## ✨ Key Features
- **Swarm Intelligence Architecture**: 10 independent AI ministers analyze situations concurrently before the President agent synthesizes a master strategy.
- **Local AI Powered**: Fully integrated with local [Ollama](https://ollama.com/) models for secure, fast, and free agentic reasoning. Auto-detects the smartest models on your machine (e.g., DeepSeek, Gemma3, Llama3).
- **Real-World Grounding**: Ingests real-time demographics and economic data directly from the **World Bank API**.
- **Live Contextual Search**: Uses `duckduckgo-search` to dynamically scrape the live internet for news and current events based on your exact simulation goal.
- **Mathematical Projections**: Produces mathematically sound JSON projections generated by the LLM, anchored to real GDP and population baselines.
- **Beautiful Premium UI**: A highly polished, responsive Light Theme dashboard featuring glassmorphism shadows and seamless WebSocket streaming.

## 🛠️ Prerequisites
- **Python 3.9+**
- **Ollama**: You must have [Ollama](https://ollama.com/) installed and running locally.
- **Ollama Models**: Pull at least one capable model. The system actively looks for advanced tags (e.g., `deepseek`, `gemma3`, `minimax`, `llama3`). 
  ```bash
  ollama run deepseek-coder  # or your preferred model
  ```

## 🚀 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Wazir.git
   cd Wazir
   ```

2. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## 🎮 Usage

1. **Ensure Ollama is running** in the background (default port `11434`).
2. **Start the FastAPI simulation server:**
   ```bash
   python -m main
   # OR
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```
3. **Open the web dashboard** in your browser:  
   👉 `http://localhost:8000`

Enter a country (e.g., "France", "Japan", "Brazil") and an optional goal (e.g., "Triple military defense spending" or "Drastically reduce inflation") and click **SIMULATE** to watch your AI cabinet go to work!

## 📁 System Architecture
- `main.py`: The FastAPI server and WebSocket hub.
- `core/simulation.py`: The async hive-mind engine orchestrating the swarm.
- `core/ollama_client.py`: The interface to your local models.
- `core/data_scraper.py`: Fetches real World Bank and live DuckDuckGo facts.
- `agents/`: Contains the specific prompts and personas of all 10 cabinet ministers.
- `static/css/style.css`: The premium light theme design parameters.
